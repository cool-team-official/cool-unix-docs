# cool-vibrate

震动效果

## 兼容性

| 平台       | 支持状态 | 说明                   |
| ---------- | -------- | ---------------------- |
| 鸿蒙 OS    | ✅ 支持  | 支持鸿蒙 OS 最新版本   |
| iOS        | ✅ 支持  | 支持 iOS 9.0+          |
| Android    | ✅ 支持  | 支持 Android 5.0+      |
| Web        | ✅ 支持  | 支持现代浏览器         |
| 微信小程序 | ✅ 支持  | 支持微信小程序最新版本 |

### 示例

```ts
import { vibrate } from "@/uni_modules/cool-vibrate";

vibrate(1);
```
